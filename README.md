# Project 1 - Web Development Basics

## Project Description
This project is a simple web-based portfolio/website developed as part of coursework. It demonstrates the use of basic HTML and CSS to create structured web pages with navigation, layout design, and content presentation.

The project focuses on building foundational skills in web development such as page structuring, styling, and linking multiple pages.

---

## SDG Theme

- Enhancing digital literacy skills

- Encouraging students to learn web development fundamentals

## Features
- Home page with navigation menu
- Multiple linked pages (Home, About, etc.)
- Simple and clean UI design using CSS
- Internal navigation between pages
- Basic responsive layout structure
- Beginner-friendly HTML & CSS implementation
